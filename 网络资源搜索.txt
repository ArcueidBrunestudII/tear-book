  3. DeepSeek AI 模型

  DeepSeek-OCR (文档识别)

  - https://github.com/deepseek-ai/DeepSeek-OCR - 官方仓库
  - https://huggingface.co/deepseek-ai/DeepSeek-OCR - 模型下载
  - https://docs.vllm.ai/projects/recipes/en/latest/DeepSeek/DeepSeek-OCR.html - 部署文档
  - https://arxiv.org/html/2510.18234v1 - 技术原理

  OCR Prompt 模板：
  - 文档转换：<image>\n<|grounding|>Convert the document to markdown
  - 普通OCR：<image>\n<|grounding|>OCR this image
  - 纯文字提取：<image>\nFree OCR

  DeepSeek-R1 (推理模型)

  - https://github.com/deepseek-ai/DeepSeek-R1 - 官方仓库
  - https://api-docs.deepseek.com/news/news250120 - 官方 API 说明
  - https://www.bentoml.com/blog/the-complete-guide-to-deepseek-models-from-v3-to-r1-and-beyond - V3/R1 对比
  - https://www.datacamp.com/tutorial/deepseek-api - 实用示例

  R1 使用建议：
  - 复杂推理任务用 R1，简单任务用 V3 省成本
  - 系统提示词在 R1-0528 版本已支持
  - 利用 context caching 降低重复查询成本
